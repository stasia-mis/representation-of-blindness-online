
Проект автоматизированного сбора и анализа дискурсивных практик конструирования агентности людей с нарушением зрения в социальной сети ВКонтакте.

## Описание

Исследование сравнивает два типа дискурса:
- **Институциональный** (публикации НКО и некомерческих проектов для слабовидящих и незрячих)
- **Индивидуальный** (личные блоги (в тч. страницы) незрячих авторов)

**Методология:** Mixed-method анализ с применением:
- Weak supervision для автоматической разметки
- Multi-label классификация (4 категории дискурса)
- Статистические тесты (Mann-Whitney U, Chi-square)
- Тематическое моделирование (LDA)

**Корпус:** 91,358 публикаций (44,529 НКО + 46,829 блогов)

**Основные находки:** Все различия между сегментами статистически значимы (p < 0.001)

##  Использование

#### 1.1. Поиск групп

```bash
python step1_parse_VKv2.py
```


**Важно:** После этого нужна **ручная сортировка** групп на НКО и личные блоги!

#### 1.2. Поиск блогеров (параллельный запуск)

```bash
python step2_blogger.py
```
**Важно:** Рекомендуется использовать 7-15 версий парсера с разными списками групп и разными сервисными токенами из-за ограничения ВКонтакте

#### 1.3.Финальный корпус

```bash
python step3_vk_corpus.py
```

**Важно:** Используется ручной список групп и пользователей фильтрация по ключевым словам не применяется
формируются итоговые файлы:
nko.csv
blogs.csv

### Этап 2: Анализ данных

#### 2.1. Основной анализ

```bash
python modelv12.py
```

**Конфигурация в коде:**
- `USE_SAMPLING = True` - быстрый анализ на выборке (5K+5K)
- `USE_SAMPLING = False` - полный корпус (91K постов)

**Результаты:**
- `output/nko_sample_predictions.csv` - предсказания для НКО
- `output/blogs_sample_predictions.csv` - предсказания для блогов
- `output/statistical_comparison.csv` - статистические тесты

#### 2.2. Валидация weak labels

```bash
python validate_weak_labels.py
```

**Workflow:**
1. Создать выборку для ручной проверки (100 постов)
2. Размечать вручную в Excel
3. Оценить качество (precision, recall, F1)
4. Оптимизировать threshold

## Дискурсивные категории

| Категория | Описание | Маркеры |
|-----------|----------|---------|
| **Victim** | Дискурс "жертвы", патернализм | страдать, нуждаться, помощь |
| **Supercrip** | Героизация, "суперкалека" | несмотря, вопреки, преодолеть |
| **Agency** | Агентность, самостоятельность | работать, учиться, создавать |
| **First_person** | Я-высказывания | я, мы, мой, наш |

## Основные результаты

### Статистические различия

| Категория     | НКО   | Блоги | Разница | p-value |
|--------------|------|------|--------|--------|
| Victim       | 14.3% | 11.0% | −3.3% | < 10⁻⁴⁷ |
| Supercrip    | 11.7% | 10.4% | −1.3% | 0.0019 |
| **Agency**   | **22.7%** | **32.7%** | **+10.0%** | < 10⁻⁷³ |
| First_person | 1.60 | 1.82 | +0.22 | < 10⁻⁸⁹ |

### Качество модели:

- **Macro F1:** 0.80 (приемлемо для weak supervision)
- **Micro F1:** 0.70 (хорошо)
- **Hamming Loss:** 0.12 (низкий) хороший результат

### Валидация:

- **Precision:** 0.54
- **Recall:** 0.82
- **F1:** 0.64 (хорошее качество для weak supervision)
